# 01 - Automatiza Routers Huawei con Python + Netmiko

Tutorial práctico donde explico paso a paso cómo automatizar la obtención de información en routers Cisco usando **Python** y las librerías **Netmiko** y **Pandas**. 

Este es el material complementario del primer video de mi canal de YouTube.

## ¿Qué se va a mostrar?

- Conexión SSH segura a routers Cisco
- Uso de Netmiko para enviar comandos y obtener información
- Automatización de obtención de información básica
- Manejo de errores
- Ejecución del script en entornos virtual usando GNS3

---

## Requisitos

- Python 3.10 o superior
- Librería `netmiko` `pandas`
- Acceso SSH a equipos Cisco (físico o en GNS3)
- Conocimientos básicos de Python (recomendado)

Instalación de la librería:
```bash
pip install netmiko
pip install pandas
